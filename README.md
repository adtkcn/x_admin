## 简介

我们希望能够为开源社区做出更多的贡献，推出永久免费开源的 x_admin 通用前后端分离管理后台系统。遵循 MIT 开源许可协议，您可以免费使用，甚至允许把你基于 x_admin 开发的软件应用开源、发布、销售。
